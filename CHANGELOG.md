# Changelog

## 1.0.1

- Added a before-and-after demo image to the English and Korean documentation.

## 1.0.0

- Initial stable release of Quiet Embed.
- Makes Markdown note embeds blend into surrounding document content by removing default embed chrome, titles, indentation, and empty top spacing.
- Adds a subtle edge highlight on hover or focus and supports source-note navigation with Command-click on macOS or Ctrl-click on Windows/Linux.
